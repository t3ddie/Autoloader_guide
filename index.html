AutoLoader v2

Quick Start Guide

The AutoLoader is an attachment for the Drywise inline filament dryer, designed to automate filament loading and continuously feed new material from the spool to your printer. This eliminates resistance from the spool, especially with heavier spools (2kg+), preventing print artifacts and underextrusions.

Setup Instructions

  1. Remove the input port cover or pre-heater from the Drywise.
  2. Connect the AutoLoader's **input part**.
  3. Remove the output port cover from the Drywise and store the original pin.
  4. Install the AutoLoader's **output part** using the longer dowel/pin provided.
  5. Using the provided screws, attach the **controller** to the front top of the Drywise.
  6. Connect the cables from the ports to the controller.
  7. Measure and cut the PTFE tubing to the required lengths (Drywise output to extruder, and Drywise input to spool).
  8. Securely connect the PTFE tubes.

Setup Tips

  • The output PTFE tube is crucial for the buffer switch to function.
  • Ensure tubes are pushed in fully.
  • Keep the path from the Drywise output to the printer as short as possible.
  • Ensure ports are not obstructed.

Controller & Usage

Press and hold a button to activate its function.

A

Auto

Automatically advances filament during printing.

L

Load

Manually loads filament into the Drywise.

U

Unload

Automatically unloads filament from the system.

Operating Procedure

Option A: Manual Operation

  1. Start a predrying cycle on the Drywise.
  2. Once complete, choose to perform an 'additional drying cycle' or 'load the printer'.
  3. If needed, use the **Load (L)** button to feed more material for the additional cycle.
  4. Select 'load the printer' on the Drywise, skipping its manual instructions.
  5. Use the **Load (L)** button to feed filament to your printer's extruder.
  6. Press and hold the **Auto (A)** button to switch to Auto mode.
  7. Load the printer normally and extrude until the filament is clean and dry.

Option B: Script-Based Operation

  1. Use the **Load (L)** button to manually load filament.
  2. Switch to Auto mode by pressing and holding the **Auto (A)** button.
  3. Load the printer normally, extruding until the filament is clean and dry.
  4. On the Drywise, skip to the printing stage.
  5. Start the print from your printer, ensuring the G-code file was sliced with the appropriate script.
